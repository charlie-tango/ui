# Theme

The `theme` is based on the
[System UI Theme Specification](https://styled-system.com/theme-specification). You can use the base
theme by importing it from the library - or you can copy the values and adapt it fit your needs.

```typescript
import { baseTheme } from '@charlietango/ui';

export const theme = { ...baseTheme };
```
