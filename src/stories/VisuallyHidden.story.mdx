# Visually Hidden

If you just set `display: none;` or `[hidden]` on elements, it will prevent it from being found by
screen readers. Instead hide the content with styling, but keep it readable for screen readers. Read
[how to hide content](https://a11yproject.com/posts/how-to-hide-content/) for more insights.

## Usage

```tsx
import { VisuallyHidden } from '@charlietango/ui';

const Example = () => <VisuallyHidden>This text can still be found</VisuallyHidden>;
```
