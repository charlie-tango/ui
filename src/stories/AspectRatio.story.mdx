# Aspect Ratio

Use the `<AspectRatio />` to force a fixed height and width ratio. This is commonly used to give
images and video a fixed size.

<Canvas>
  <AspectRatio ratio={16 / 9}>
    <AspectRatioItem as={Item}>16:9 Aspect Ratio</AspectRatioItem>
  </AspectRatio>
</Canvas>

## Usage

```jsx harmony
import { AspectRatio, AspectRatioItem } from '@charlietango/ui';

export const MyComponent = (props) => (
  <AspectRatio ratio={16 / 9}>
    <AspectRatioItem>16:9 Aspect Ratio</AspectRatioItem>
  </AspectRatio>
);
```
