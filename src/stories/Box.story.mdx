# Box

The `<Box />` is the building block that gives us the features of `styled-system`. It's a
reimplementation of the of the [Rebass Box](https://rebassjs.org/box), and should be used in the
same manner.

## Usage

```jsx harmony
import { Box } from '@charlietango/ui';
```

### Using props to style

<Preview>
  <Box p={5} fontSize={4} w={[1, 1, 0.5]} color="white" bg="blue">
    Box
  </Box>
</Preview>

### Using the `sx` prop

The `sx` property allows you to apply styles from a CSS object, while still being able to access the
[styled-system](https://styled-system.com/) theme values. So you can use arrays to define the
responsive values.

Note that it doesn't have all of the same transformation methods, so some values will be transformed
differently from setting the value on the `<Box / >`. As an example, the `width` property doesn't
transform values between 0 and 1 to percentages.

<Preview>
  <Box
    sx={{
      p: 5,
      fontSize: 4,
      width: ['100%', null, '50%'],
      color: 'white',
      bg: 'blue',
    }}
  >
    Box
  </Box>
</Preview>
