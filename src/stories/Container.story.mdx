# Container

The `<Container />` component is used to limit the maximum width of your content, while keeping it
centered on the page.

## Configuration

The max width for the `<Container />` can be controlled by setting the `sizes.container` value on
the theme.

```js
const theme = {
  sizes: {
    container: 1280,
  },
};
```

### Variants

Container variants can be defined in the `theme.layout` object. The Container component uses
`theme.layout.container` as its default variant style. Like all other styles, values can be
responsive. Use to this change the padding for various breakpoints.

```js
const theme = {
  layout: {
    container: {
      px: [3, 4, null, 6],
    },
  },
};
```

### API

The components inherit the [Box](/?path=/docs/components-box--page#props) props. It currently
doesn't have any special extra props.
