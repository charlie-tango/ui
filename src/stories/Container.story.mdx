# Container

The `<Container />` component is used to limit the maximum width of your content, while keeping it centered on the page.

## Configuration

The default styling for the `<Container />` can be controlled by setting the values on the theme.
Use this to provide the `max-width` and `padding` values you are generally using on the site.

```js
const theme = {
  variants: {
    container: {
      maxWidth: 1200,
      px: [3, 4, null, 6],
    },
  },
}
```

Like all other styles, values can be responsive. Use to this change the padding for various breakpoints.
